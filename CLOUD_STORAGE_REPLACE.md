# 云存储提供商替换说明

## 当前状况分析

### 为什么不能直接替换为百度网盘和阿里网盘？

1. **技术限制**
   - Uppy 官方不提供百度网盘和阿里网盘的插件
   - 这两个平台没有开放标准的 OAuth 接口供第三方集成
   - 缺乏官方的 API 文档支持文件选择功能

2. **替代方案建议**

#### 方案一：使用支持的国内云服务
```
原服务 → 替代服务
Google Drive → 腾讯云 COS
Dropbox → 阿里云 OSS
OneDrive → 七牛云存储
```

#### 方案二：强化本地文件选择
- 保持现有的国际云存储服务作为可选项
- 重点优化本地文件选择体验
- 添加拖拽上传功能增强

#### 方案三：自定义集成（高级）
需要开发：
- 自定义 OAuth 认证流程
- 文件列表获取 API
- 文件下载接口
- 上传进度监控

## 推荐实施策略

### 短期方案（立即可用）
保留现有架构，添加说明提示用户使用本地文件选择为主要方式。

### 中期方案（1-2周开发）
集成腾讯云 COS 和阿里云 OSS 插件。

### 长期方案（1-2月开发）
开发自定义的百度网盘和阿里网盘集成插件。

## 当前功能状态

✅ Google Drive - 正常工作（需要 Companion 服务）
✅ Dropbox - 正常工作（需要 Companion 服务）  
✅ OneDrive - 正常工作（需要 Companion 服务）
❌ 百度网盘 - 无官方支持
❌ 阿里网盘 - 无官方支持
